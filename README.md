## 🚀 About The Project

**DhanGuru** is a smart personal finance management system designed to help users track expenses, manage budgets, and make better financial decisions.

It simplifies money management by giving insights into spending patterns and helping you stay in control of your finances.

---

## ✨ Features

* 💸 Track daily expenses
* 📊 Visualize spending patterns
* 🧾 Categorize transactions
* 🎯 Budget management
* 📈 Financial insights & analytics
* 🔒 Simple and secure usage
* ⚡ Lightweight and fast

---

## 🧠 How It Works

* 📥 Input your income and expenses
* 🗂️ Categorize transactions
* 📊 Analyze spending trends
* 🎯 Set and track budgets
* 📈 Improve financial habits
